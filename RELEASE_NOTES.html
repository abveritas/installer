Build-Notes

Log Packages

Now you can install log-packages for some of the packages in our desktop repository. Just type this cmd: sudo pacman -Sy chakra-log

VGA_SWITCHEROO and ACPI_CALL

With this release we support vga_switcheroo and acpi_call.
Please help us with testing laptops with hybrid-cards. To test if you can switch on/off your discrete graphics card using "test_off.sh".
For more info about hybrid graphics consult: linux-hybrid-graphics.blogspot.com.

System Requirements

  • a Intel Pentium II or higher processor or AMD64-ready CPU for 64bit
  • 384MiB of RAM for running the live system and 512MiB of RAM for the installation
  • around 3 Gib of available hard disk space (the more the better)
  • a screen capable of 800 x 600 or higher resolution
  • a reasonably fast internet connection, e.g. no modem (would be too slow)
